### 2.3 ONE-WAY FUNCTIONS

- One way functions are essential to many cryptographic protocols
- They are easy to compute, but difficult to reverse
- Mathematically there is not proof that one-way functions exist
- Trapdoor one-way functions have more efficient use in cryptography as secret additional info is available to intended recipients which will help in reversing the function


https://learning.oreilly.com/library/view/applied-cryptography-protocols/9781119096726/10_chap02.html#chap02-sec003




### 2.4 ONE-WAY HASH FUNCTIONS

- A function that takes a pre-image and converts it to a hash value
- One-way hash functions are easy to compute, but hard to reverse to generate a specific pre-image
- A good one-way hash function is collision-free, meaning it is hard to generate multiple pre-images from a hash value
- Despite the hash function being public, it is impossible to reverse output to obtain the pre-image


https://learning.oreilly.com/library/view/applied-cryptography-protocols/9781119096726/10_chap02.html#chap02-sec003
